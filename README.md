# ML-Based Skin Cancer Classifier

An intelligent medical image analysis system that classifies skin lesions and moles using machine learning and deep learning techniques.

## Features
- **CNN-based Classification**: High-accuracy classification using Convolutional Neural Networks.
- **Traditional ML Support**: Manual feature extraction (ABCDE criteria) with SVM and Random Forest.
- **Interactive Dashboard**: Sleek web interface for image uploads and real-time predictions.
- **Comprehensive Analytics**: Visualization of model performance, including confusion matrices and accuracy curves.

## Getting Started

### Prerequisites
- Python 3.9+
- Pip

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Project Structure
- `data/`: Datasets and processed images.
- `models/`: Trained model files.
- `notebooks/`: Data exploration and experimentation.
- `src/`: Core logic for preprocessing, training, and API.
- `web/`: Frontend dashboard.
